# Smart Salary 2026

![Smart Salary 2026](https://img.shields.io/badge/Status-Premium-emerald)
![Language](https://img.shields.io/badge/Lang-RU%20%7C%20KK-blue)

**Smart Salary 2026** — это высокотехнологичный персональный калькулятор для расчета заработной платы и планирования бюджета, специально адаптированный под производственный календарь Казахстана на 2026 год.

## ✨ Особенности

-   **Премиальный UX/UI**: Дизайн в стиле Digital Luxury с использованием Obsidian & Emerald палитры и Glassmorphism эффектов.
-   **Локализация (RU/KZ)**: Полная поддержка русского и казахского языков с мгновенным переключением.
-   **Точные расчеты 2026**: Автоматический учет МРП, налогов (ОПВ, ИПН, ВОСМС), праздничных выплат и ночных смен.
-   **Умное планирование**: Модуль анализа расходов с визуальной диаграммой (Donut Chart) и панелью экономии.
-   **11+ Графиков работы**: Поддержка различных сменных графиков (2/2, 3/1, дневные, ночные и т.д.) с возможностью ручной правки.
-   **Высокая производительность**: Плавные анимации на базе движка GSAP.

## 🛠 Технологии

-   **Structure**: HTML5 (Semantic)
-   **Styling**: Vanilla CSS (Variables, Grid, Flexbox)
-   **Logic**: Pure JavaScript (ES6+)
-   **Animations**: GSAP (GreenSock Animation Platform)

## 🚀 Запуск

Просто откройте `index.html` в любом современном браузере. Приложение работает полностью на стороне клиента и не требует сервера.
